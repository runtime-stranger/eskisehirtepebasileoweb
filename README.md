# Eskişehir Tepebaşı Leo Kulübü Web Sitesi

Eskişehir Tepebaşı Leo Kulübü (118-K Yönetim Çevresi) için hazırlanmış, modern ve kurumsal bir web sitesi projesidir.

## 🔗 Canlı Site Bağlantısı
Projenin canlı haline aşağıdan ulaşabilirsiniz:
https://eskisehirtepebasileo.org/
https://runtime-stranger.github.io/eskisehirtepebasileoweb/

## 👤 Müşteri Analizi ve Proje Amacı
* **Müşteri:** Eskişehir Tepebaşı Leo Kulübü (Uluslararası Lions Kulüpleri Birliği'nin gençlik yapılanması).
* **Hedef Kitle:** Sosyal sorumluluk projelerine ilgi duyan gençler, potansiyel üyeler ve bağışçılar.
* **Amaç:** Kulübün kurumsal kimliğini yansıtmak, faaliyetlerini duyurmak ve dijital görünürlüğünü artırmak.

## 🛠 Kullanılan Teknolojiler
* **HTML5:** Semantik etiketler ile SEO dostu ve erişilebilir yapı.
* **CSS3:** Flexbox yerleşimleri, özel CSS değişkenleri (Variables) ve Dark Mode desteği.
* **JavaScript:** Çoklu dil (TR/EN) seçeneği ve kullanıcı deneyimini artıran scroll animasyonları.
* **Araçlar:** Geliştirme sürecinde OpenCode ve yapay zeka kodlama asistanlarından yararlanılmıştır.

## 📁 Proje Yapısı ve Commit Özetleri
Proje, hocanın belirlediği kriterlere uygun olarak anlamlı bir klasör yapısıyla kurgulanmış ve adım adım commit edilmiştir:

1.  `feat`: Ana semantik iskelet ve SEO meta etiketleri.
2.  `style`: `css/` klasörü altında modern ve responsive tasarım dosyaları.
3.  `js`: `js/` klasörü altında dinamik dil ve animasyon özellikleri.
4.  `assets`: `images/` klasörü altında optimize edilmiş görsel materyaller.
5.  `docs`: Proje raporu ve bu README dökümantasyonu.
